# sk-clinic.ru

Официальный сайт клиники SK Clinic на базе WordPress.

## Описание

Проект представляет собой полнофункциональный WordPress сайт для медицинской клиники. Сайт включает в себя все необходимые функции для представления услуг клиники, записи на прием, публикации новостей и статей о здоровье.

## Технические характеристики

- **CMS**: WordPress
- **Тема**: Astra (кастомная настройка)
- **PHP версия**: 7.4+
- **MySQL версия**: 5.7+

## Структура проекта

```
sk-clinic.ru/
├── wp-admin/          # Административная панель WordPress
├── wp-content/        # Контент сайта
│   ├── themes/        # Темы оформления
│   │   └── astra/     # Тема Astra
│   └── plugins/       # Плагины
├── wp-includes/       # Ядро WordPress
└── wp-config.php      # Конфигурационный файл
```

## Установка

### Требования

- PHP 7.4 или выше
- MySQL 5.7 или выше
- Apache/Nginx веб-сервер
- Модули PHP: mysqli, curl, mbstring, xml, zip

### Шаги установки

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/evschankin555/sk-clinic.ru.git
   cd sk-clinic.ru
   ```

2. **Настройте базу данных**
   - Создайте базу данных MySQL
   - Импортируйте дамп базы данных (если есть)

3. **Настройте wp-config.php**
   - Скопируйте `wp-config-sample.php` в `wp-config.php`
   - Укажите данные для подключения к базе данных:
     ```php
     define('DB_NAME', 'your_database_name');
     define('DB_USER', 'your_database_user');
     define('DB_PASSWORD', 'your_database_password');
     define('DB_HOST', 'localhost');
     ```

4. **Настройте права доступа**
   ```bash
   chmod 644 wp-config.php
   ```

5. **Завершите установку через веб-интерфейс**
   - Откройте сайт в браузере
   - Следуйте инструкциям мастера установки WordPress

## Разработка

### Локальная разработка

1. Установите локальный сервер (XAMPP, WAMP, Local by Flywheel и т.д.)
2. Настройте виртуальный хост для домена `sk-clinic.ru.local`
3. Клонируйте репозиторий в директорию сервера
4. Настройте базу данных и `wp-config.php`

### Работа с темой

Тема Astra находится в `wp-content/themes/astra/`. Для кастомизации:

- Создайте дочернюю тему (рекомендуется)
- Или редактируйте файлы напрямую (не рекомендуется для продакшена)

## Деплой

### Автоматический деплой

Проект настроен на автоматический деплой через GitHub webhook:

- При каждом push в ветку `main` происходит автоматическое обновление на сервере
- Webhook сервер запущен как systemd сервис

### Ручной деплой

1. Подключитесь к серверу по SSH
2. Перейдите в директорию проекта:
   ```bash
   cd /var/www/sk-clinic.ru
   ```
3. Обновите код:
   ```bash
   git pull origin main
   ```
4. Обновите базу данных (если нужно):
   ```bash
   wp db upgrade
   ```
